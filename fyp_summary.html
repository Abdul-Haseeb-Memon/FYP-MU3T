Project Abstract
What this project is about

This project builds a comprehensive security framework to protect Agentic AI systems (AI agents that take real-world actions using LangChain and LLMs) from Prompt Injection Attacks — a critical vulnerability where malicious instructions override an AI agent's original rules. The framework operates in three layers: a Detection Layer that identifies attacks using keyword filtering, semantic AI analysis, and vector database similarity search; an Isolation Layer that uses prompt sandwiching to prevent instruction hijacking; and a Prevention Layer that sanitizes AI outputs before delivery. The system will be evaluated against 300+ attack prompts, targeting 90%+ detection accuracy with below 10% false positive rate, demonstrated through a live Streamlit web interface.

Research Novelty & Contribution
What makes this project new and original

Unified 3-Layer Framework

No existing solution combines detection + isolation + prevention in one unified system. This is the gap we fill.

AI vs AI Detection

We use an AI model (Llama3) to detect AI attacks — a novel approach where semantics are understood, not just keywords.

Both Attack Types Covered

Most solutions handle only direct attacks. Our framework handles both direct AND indirect prompt injection attacks.

First at MUET

No research group at MUET Jamshoro has published on this topic. This is a completely fresh research contribution for the university.
